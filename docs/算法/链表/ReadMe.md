链表篇正式结束，来总结一下。

首先链表里面一定要画出具体执行流程。

其次链表常用的方法有双指针，递归，迭代等操作。

修改链表有两种方式，一种是添加虚拟节点，另一个是在原链表的基础上进行操作。

反转链表有两种方式，一个是头插法，另一个是穿针引线法，一定需要画出流程图，不然中间节点的关系就不知道了。
